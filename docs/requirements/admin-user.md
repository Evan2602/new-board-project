
## 1. 회원 관리

### 1.1 회원 목록 조회
- 회원 목록 조회 API 제공
- 조회 항목:
    - 회원 ID
    - 이메일
    - 닉네임
    - 가입일
    - 상태 (정상, 정지, 탈퇴)
    - 마지막 로그인 시각
- 검색 조건:
    - 이메일
    - 닉네임
- 정렬:
    - 가입일 최신순 / 오래된 순

### 1.2 회원 상세 조회
- 특정 회원 상세 정보 조회
- 회원이 작성한 게시글 목록 조회
- 최근 로그인 이력 조회

### 1.3 회원 상태 변경 (제재)
- 회원 상태를 정상 / 정지 / 탈퇴로 변경 가능
- 상태 변경 시 사유를 함께 저장한다.
- 제재 이력 관리:
    - 변경 전 상태
    - 변경 후 상태
    - 사유
    - 처리 관리자
    - 처리 시각

### 1.4 강제 로그아웃
- 특정 회원의 모든 인증 토큰을 만료 처리하여 강제 로그아웃 가능해야 한다.

### 1.5 비밀번호 초기화 (선택)
- 관리자가 특정 회원의 비밀번호를 임시 비밀번호로 초기화할 수 있다.
- 초기화 이력은 로그로 남긴다.
