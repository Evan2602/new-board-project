# 백엔드 수정 완료 안내

## 요청하신 `userName` 필드 추가 작업이 완료되었습니다.

---

## 변경된 응답 형식

`BoardResponse`를 반환하는 모든 엔드포인트에 `userName` 필드가 추가되었습니다.

```json
{
    "id": 1,
    "title": "제목",
    "content": "내용",
    "authorId": "hong123",
    "userName": "홍길동",
    "createdAt": "2026-02-26T12:00:00",
    "updatedAt": "2026-02-26T12:00:00"
}
```

---

## 적용된 엔드포인트

| 엔드포인트         | 메서드 | 설명             |
| ------------------ | ------ | ---------------- |
| `/api/boards`      | GET    | 게시글 목록 조회 |
| `/api/boards/{id}` | GET    | 게시글 상세 조회 |
| `/api/boards`      | POST   | 게시글 생성      |
| `/api/boards/{id}` | PUT    | 게시글 수정      |

---

## 참고 사항

- `authorId` 필드는 **기존 그대로 유지**됩니다. 삭제되지 않았으니 기존 로직에 영향 없습니다.
- `userName`은 회원가입 시 입력한 **닉네임**입니다.
- 탈퇴 등의 이유로 사용자 정보를 찾을 수 없는 경우, `userName`은 `authorId` 값으로 대체됩니다.
